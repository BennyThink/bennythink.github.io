YouTube视频的下载方法

 

 

YouTube上的视频质量可以说是上乘中的上乘。国际标准的720P、1080P甚至是2K、4K，哪像国内的网站动不动就所谓的“高清”、“超清”，说不定原来码率压缩的有多大。而且YouTube的部分视频还带有字幕哦。

既然视频质量怎么好，怎么可以不收藏呢！

下面简要介绍几个下载YouTube视频的方法

需要准备的工具有：百度云账号、科学上网的梯子（如果没有梯子，请看此博文）。

 

 

方案一．在线解析下载：

以clipconverter为例。

首先，你要去油管找你想要下载的视频，然后复制它的链接。我随意找了个视频，链接如下https://www.youtube.com/watch?v=sFCJx9N61kE

之后打开http://www.clipconverter.cc/，粘贴视频网址，点击continue

选择解析度，然后点击下面的start，

 

然后果断的点download吧！

别急，这么慢的速度怎么受得了呢？使用Chrome的用户可以按Ctrl+J打开下载管理，然后选择“复制链接地址”

打开百度云，离线下载，粘贴进地址，开始下载，等着中转完成吧！

同样的道理，有兴趣的小伙伴也可以试试用迅雷离线等。

另外，如果有自己的VPS，wget 下载链接，做中转也是一个不错的选择。日韩的VPS可以尝试下。

如果此方法不奏效，请从头到尾再来一次！

另外推荐个在线解析YouTube视频的网站，

http://www.findyoutube.com/

 

方案二．专用下载器

硕鼠YouTube视频下载器

使用方法也很简单，设置代理，新建任务，粘贴地址，确定即可。如果使用的工具是SOCKS5代理（比如说Tor，Shadowsocks，请记得勾选SOCKS5选项）

 

其他类似工具：Clipgrab

 

方案三．各种Linux发行版、Mac OS

下载个视频都来敲终端，那感觉是该有多好！

2016年4月20日更新 此工具也可以下载国内视频网站的视频了！

很幸运的是Ubuntu下有一个叫youtube-dl的基于python的小工具

基本使用方法是先安装

sudo apt-get install youtube-dl

之后再

youtube-dl VIDEO_URL

VIDEO_URL为视频的地址。

 

可是我想下1080P的！没关系，先用F参数看下这个视频都有啥分辨率的啊

youtube-dl -F http://www.youtube.com/watch?v=BlXaGWbFVKY

分辨率前面有一个标号，那么我们就指定这个！

youtube-dl -f 17 http://www.youtube.com/watch?v=BlXaGWbFVKY

更详细的教程看此处：https://linux.cn/article-6473-1.html
